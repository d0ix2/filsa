# filsa

작가 사후 70년이 지나 저작권이 만료된 문학 작품을 이용해, 문학 작품에서 랜덤으로 한 문단을 가져와 필사 학습할 수 있도록 제공합니다. 한국 문학 작품, 일본 문학 작품을 제공합니다.  
著作権が作者の死後70年を経て消滅した文学作品を活用し、ランダムに選ばれた一節を提供することで、筆写学習ができるサービスです。韓国文学作品および日本文学作品を対象としています。
